Setmark — wireframes v2

Revision · RIR · rest-is-the-log · NEXT-first progression
What changed from v1. RIR replaces RPE everywhere. Rest screen now shows the just-completed set as an editable row. New catch-up flow handles "I forgot to start/stop" inline — no split-timer UI. Supersets, circuits, for-time collapse to a single giant NEXT button; logging is deferred to rest or completion. AMRAP is +1 round only, with a partial-rep picker at the end. Completion is a ledger you can tap into (any group, any set editable) with a voice-dictated note. Watch always shows HR + time; one tap = progress. Swap = longpress (sheet alt kept for discoverability). Offline demoted to a tiny corner pill.
