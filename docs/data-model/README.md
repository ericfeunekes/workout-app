# Data Model

This folder summarizes the current data model decisions and the intended evolution path.
It is intentionally minimal and points to the deeper reference in `docs/roadmap/appendix-a-data-model.md`.

## Sections
- `overview.md` — The layers and how they connect.
- `catalog.md` — Exercises, muscles, movement patterns, and equipment.
- `library.md` — Raw ingestion and workout templates.
- `planning.md` — Goals, schedules, and calendar linkage.
- `execution.md` — Sessions and logging.
- `intent.md` — Intent taxonomy and stimulus labeling.
- `examples.md` — YAML examples for intent + metadata.
