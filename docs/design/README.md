# Setmark — start here

This project is a design handoff for a mobile workout logger.

**Read `HANDOFF.md` first** — it's the index and tells you where to go next.

## Quick map

- `HANDOFF.md` — index · read first
- `BACKLOG.md` — what's done, what's next, open questions
- `FLOWS.html` — visual map of every screen (open in browser)
- `RULES.md` — behavior/logic rules (autoreg, persistence, etc.)
- `WorkoutDB Hi-Fi.html` — the legacy-named hi-fi interactive prototype (one flow)
- `WorkoutDB Wireframes v2.html` — legacy-named wireframes for every flow
- `src/`, `components/`, `styles/` — source for the above
- `_archive/` — superseded work · ignore unless doing archaeology
