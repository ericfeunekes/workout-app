# Shared Schema

Source of truth for cross-stack data contracts between `server/` (Python) and `app/` (Swift).

## What's here

- **`openapi.json`** — committed OpenAPI 3.1 schema exported from the FastAPI app. The authoritative wire contract. Regenerated by the contract tests; drift fails the build.
- **`Package.swift` + `Sources/WorkoutDBSchema/`** — hand-written Swift Codable DTOs that mirror the server's Pydantic request/response schemas. Consumed by the iOS app via SwiftPM.
- **`Tests/WorkoutDBSchemaTests/`** — `swift test` round-trip + enum-parity tests.

## Approach

**Hand-written DTOs on each side, cross-checked by contract tests.** Not code-gen from OpenAPI. Rationale:

- Single-user repo with cutover philosophy — duplication is cheap to maintain, and hand-written types are clearer to read.
- Contract tests in `tests/contract/` fail the build on drift, so the "keep in sync manually" cost is enforced.
- Code generation (openapi-generator, swift-openapi-generator) adds a build step and usually produces clunky types.

If the entity count explodes or drift becomes a maintenance burden, revisit — switching to codegen is a cutover, not a gradual migration.

## Regenerating `openapi.json`

Automatic on test run. To update manually:

```bash
WORKOUTDB_BEARER_TOKEN=dummy WORKOUTDB_DB_PATH=/tmp/dummy.db \
  uv run python -c "import json; from workoutdb_server.main import app; print(json.dumps(app.openapi(), indent=2))" \
  > schema/openapi.json
```

The contract test `tests/contract/test_openapi_drift.py` runs this and fails if the checked-in file doesn't match. Any API change must regenerate + commit the schema.

## When Swift and Python disagree

The contract tests will catch it. The flow when that happens:

1. Read the contract test failure — it names the drift (missing field, enum mismatch, type change).
2. Decide which side is correct (almost always: the side you just changed).
3. Update the other side to match in the same commit.
4. Regenerate `openapi.json` if the change was server-side.

Do not loosen the contract test to make a drift pass. The whole point of this surface is to prevent exactly that.
